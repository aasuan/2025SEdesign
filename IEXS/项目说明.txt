智慧考试系统 - 最小可运行版本 (MVP)

===========================================
已完成的功能模块
===========================================

✅ 1. 用户模块
   - 用户注册
   - 用户登录（基于Session）
   - 用户信息查询

✅ 2. 题目管理模块（老师）
   - 添加题目（支持单选、多选、判断、简答）
   - 查询题目列表
   - 更新题目
   - 删除题目

✅ 3. 考试管理模块（老师）
   - 创建考试
   - 设置考试题目
   - 查询自己创建的考试

✅ 4. 考试参与模块（学生）
   - 查看可参加的考试
   - 参加考试
   - 开始答题
   - 获取题目列表
   - 提交答案
   - 交卷并自动评分

✅ 5. 基础功能
   - RESTful API设计
   - 统一返回格式（JsonResult）
   - 基于角色的权限控制
   - Session管理
   - Redis配置（预留）

===========================================
数据库表说明
===========================================

1. user - 用户表
   - id, username, password, real_name, email, phone
   - role: 0-学生, 1-老师, 2-管理员

2. question - 题目表
   - id, title, type, option_a/b/c/d, correct_answer
   - score, difficulty, creator_id

3. exam - 考试表
   - id, exam_name, description, duration, total_score
   - start_time, end_time, creator_id, status

4. exam_question - 考试题目关联表
   - id, exam_id, question_id, order_num

5. exam_room - 考场表（学生参加考试的记录）
   - id, exam_id, student_id, status, start_time
   - submit_time, score

6. exam_record - 考试记录表（学生答题详情）
   - id, exam_room_id, question_id, student_answer, score

===========================================
API接口列表
===========================================

用户相关 (/api/user)
- POST /register - 用户注册
- POST /login - 用户登录
- POST /logout - 用户退出
- GET /info - 获取当前用户信息

题目相关 (/api/question) - 需要老师权限
- POST /add - 添加题目
- GET /list - 查询题目列表
- GET /{id} - 查询题目详情
- POST /update - 更新题目
- POST /delete/{id} - 删除题目

考试相关 (/api/exam)
- POST /create - 创建考试（老师）
- GET /list - 查询考试列表
- GET /available - 查询可参加的考试（学生）
- GET /{id} - 查询考试详情

考场相关 (/api/examRoom) - 学生功能
- POST /join - 参加考试
- POST /start/{examRoomId} - 开始答题
- GET /questions/{examRoomId} - 获取题目列表
- POST /submitAnswer - 提交答案
- POST /submit/{examRoomId} - 交卷
- GET /myExams - 查询我的考试记录

===========================================
注意事项
===========================================

1. 当前密码为明文存储，生产环境必须加密
2. Session存储在内存中，重启会丢失，建议使用Redis存储
3. 考试时间判断需要前后端配合
4. 当前自动评分仅支持客观题（单选、多选、判断）
5. 简答题需要手动评分（可后续扩展）
