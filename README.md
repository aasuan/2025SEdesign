面向 2025 软工课设的前端项目，包含本地运行与 AI Studio 部署所需配置。

## 本地运行

前置条件：安装 Node.js。

1. 安装依赖：`npm install`
2. 在 `.env.local` 配置 `GEMINI_API_KEY`（用于 Gemini 服务）
3. 启动开发服务器：`npm run dev`

## AI Studio

在 AI Studio 查看应用：https://ai.studio/apps/drive/1dGvEwKYspFameqvGQBlNU2XAfksD-gCX

如需部署，请按本地运行步骤准备依赖和环境变量，再根据 AI Studio 指引发布。
